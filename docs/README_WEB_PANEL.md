## 🔧 عیب‌یابی سریع

### مشکل: پنل باز نمیشه

**راه‌حل 1**: بررسی وضعیت
```batch
check_web_panel_status.bat
```

**راه‌حل 2**: راه‌اندازی مجدد
```batch
DEPLOY_WEB_PANEL.bat
```

**راه‌حل 3**: بررسی فایروال
```bash
ssh root@195.248.243.122
ufw allow 5000/tcp
```

### مشکل: خطای "Address already in use"

```bash
ssh root@195.248.243.122
pkill -9 -f web_api.py
cd "/root/card game"
nohup python3 web_api.py > web_panel.log 2>&1 &
```

### مشکل: خطای "ModuleNotFoundError"

```bash
ssh root@195.248.243.122
cd "/root/card game"
pip3 install flask flask-cors
```
