# 🎯 راهنمای استفاده از پنل ادمین

## 🌐 دسترسی به پنل
**آدرس**: http://localhost:5000

## 📋 مراحل استفاده:

### 1️⃣ باز کردن پنل
- مرورگر خود را باز کنید
- به آدرس http://localhost:5000 بروید
- صفحه پنل ادمین بارگیری می‌شود

### 2️⃣ تنظیمات بازی (⚙️)
- روی دکمه **"⚙️ تنظیمات بازی"** کلیک کنید
- بخش تنظیمات باز می‌شود
- **تعداد جان روزانه** را تغییر دهید (1 تا 50)
- روی **"✏️ ویرایش تنظیمات"** کلیک کنید
- تغییرات را اعمال کنید
- روی **"✅ ذخیره تغییرات"** کلیک کنید

### 3️⃣ مدیریت Cooldown (❄️)
- روی دکمه **"❄️ مدیریت Cooldown"** کلیک کنید
- لیست کارت‌های Epic و Legend نمایش داده می‌شود
- برای هر کارت:
  - روی **"✏️ ویرایش"** کلیک کنید
  - **حد مجاز برد** را تغییر دهید
  - **مدت Cooldown** (ساعت) را تنظیم کنید
  - کارت را **فعال/غیرفعال** کنید
  - روی **"✅ ذخیره"** کلیک کنید

### 4️⃣ مدیریت کارت‌ها
- **افزودن کارت جدید**: روی دکمه آبی کلیک کنید
- **ویرایش کارت**: روی دکمه ویرایش کنار هر کارت
- **حذف کارت**: روی دکمه قرمز حذف

## 🔧 تنظیمات پیش‌فرض:

### سیستم جان:
- **جان روزانه**: 10 جان
- **قوانین کم شدن**:
  - Legend شکست از Normal: -2 جان
  - Normal شکست از Legend: 0 جان
  - سایر موارد: -1 جان

### Cooldown کارت‌ها:
- **Epic**: 3 برد، 24 ساعت
- **Legend**: 2 برد، 48 ساعت

## ❗ نکات مهم:
- پنل فقط در localhost قابل دسترسی است
- تغییرات بلافاصله اعمال می‌شوند
- برای ریست همه Cooldown ها از دکمه "🗑️ ریست همه" استفاده کنید
- اگر صفحه بارگیری نشد، F5 بزنید

## 🆘 عیب‌یابی:
1. **صفحه بارگیری نمی‌شود**: 
   - بررسی کنید web API در حال اجرا باشد
   - آدرس را دوباره تایپ کنید

2. **دکمه‌ها کار نمی‌کنند**:
   - صفحه را refresh کنید (F5)
   - Console مرورگر را بررسی کنید

3. **تغییرات ذخیره نمی‌شوند**:
   - اتصال اینترنت را بررسی کنید
   - دوباره تلاش کنید
